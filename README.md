# Netfliex-Content-Analysis-Dashboard
PowerBI Deshbord Analyzing Netfliex movies &amp; Tv Shows Content Distribution Across:
-Region/contry
-Audiance Age Group
-Release year trends
-Genres & categories

-->Key Insights:-
- Netflix library contains **5185 Movies** and **147 TV Shows**
- The majority of content is targeted toward **Adult** audiences
- Content additions increased significantly between **2016 - 2020**
- **United States** and **India** contribute the highest number of titles

--> Tools Used
- Power BI
- Power Query
- DAX
- Pandas (Data Cleaning in Python, if applicable)
